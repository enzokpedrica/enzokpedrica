## 🎯 Projetos em Destaque

### 🪵 CoreWood
CoreWood é a plataforma que desenvolvi para automatizar o fluxo CAD-to-CNC na indústria moveleira. Com um único cadastro, o sistema converte arquivos STEP, gera documentação de furação e usinagem, define o roteiro de peças e produz o código CNC pronto para a máquina — reduzindo em até 97% o tempo de programação.

### 🖨️ FastPrint
Sistema de impressão automatizada de PDFs para o setor de engenharia.
Resultado: **96% de redução de clique e 83% de redução em pesquisa de documentos**.
