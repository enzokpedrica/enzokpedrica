## 🎯 Projetos em Destaque

### 🪵 CoreWood
Plataforma que automatiza o fluxo CAD-to-CNC na indústria moveleira. Com um único cadastro, o sistema converte arquivos STEP, gera documentação de furação e usinagem, define o roteiro de peças e produz o código CNC pronto para a máquina.

Frentes validadas em POC:
- **Roteirização** — definição automática do roteiro de fabricação das peças a partir do cadastro
- **Fluxo de Furação** — extração da geometria de furação diretamente do arquivo STEP e geração da documentação correspondente
- **Exportação PDF** — documentação técnica automatizada para engenharia e produção
- **Paletização** — otimização do plano de paletização das peças
  
Stack: FastAPI · React · PostgreSQL · Docker · pythonOCC

### 📁 PLM Linea
Sistema de gestão de documentos de engenharia que substituiu planilhas Excel no controle de **mais de 12.000 documentos técnicos**, com rastreabilidade e versionamento.

### 🖨️ FastPrint
Sistema de impressão e rastreamento automatizado de PDFs para o setor de engenharia, eliminando etapas manuais de busca e envio de documentos.
**Resultado: 83% de redução no tempo de pesquisa de documentos.**
